# String Methods

## camelCase

> Converts string to camel case. https://developer.mozilla.org/en-US/docs/Glossary/Camel_case

**Since**: _0.1.0_

**Arguments**

<DocsTable
  datas={[
    { param: 'str', type: 'string', description: 'The string to convert.' },
  ]}
/>

**Returns**

<DocsTable
  datas={[{ type: 'string', description: 'Returns the camel cased string.' }]}
/>

**Declaration**

```ts
declare function camelCase(str?: string): string;
```

**Examples**

```js
camelCase('foo bar'); // 'fooBar'
camelCase('foo-bar'); // 'fooBar'
camelCase('FOO__bar'); // 'fooBar'
```
