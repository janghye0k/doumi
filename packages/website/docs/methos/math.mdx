**Arguments**

<DocsTable
  datas={[

    { param: 'start', type: 'number', description: 'The start of the range.' },
    { param: 'end', type: 'number', description: 'The end of the range.' },
  ]}
/>

**Returns**

<DocsTable
  datas={[
    {
      type: 'boolean',
      description: 'Returns `true` if number is in the range, else `false`.',
    },
  ]}
/>

**Declaration**

